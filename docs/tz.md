# Техническое задание на 23.04

### Функциональные требования

Приложение позволяет:

* регистрироваться и входить в аккаунт

* создавать команду

* добавлять участников в команду

* удалять участников в команде

* создавать чаты

* отправлять сообщ\\файлы

* просматривать список участников команды

* удалять или редактировать свои сообщения

### Нефункциональные требования

* приложение должно иметь удобный интерфейс

* данные пользователей должны сохраняться в базе данных

* открывается в современных браузерах

### Сценарий использования

1. пользователь открывает сайт

2. регистрируется\\входит в систему

3. создаёт команду или присоединяется к существующей

4. открывает командный чат

5. отправляет сообщения и файлы

6. смотрит историю переписки

7. следит за этапами выполнения проекта
