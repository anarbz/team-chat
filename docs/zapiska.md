Пользователи могут регистрироваться, входить в аккаунт, создавать личные и групповые чаты, просматривать список своих чатов, редактировать состав участников, отправлять сообщения и прикреплять файлы.

Приложение написано на Python с использованием фреймворка Flask. Для авторизации используется Flask-Login, для форм - Flask-WTF, для работы с основной базой данных - SQLAlchemy. Интерфейс построен на HTML-шаблонах Jinja2, Bootstrap и собственном CSS.

**Основная Структура Проекта**

- main.py - создание Flask-приложения, подключение базы данных и маршрутов
- routes/ - обработчики страниц: авторизация, список чатов, чат, участники, редактирование и удаление
- data/ - модели основной базы данных
- services/ - логика создания чатов и работы с сообщениями
- utils/ - вспомогательные функции для баз чатов и загрузки файлов.
- templates/ - шаблоны страниц.
- static/css/ - стили интерфейса
- uploads/ - загруженные пользователями файлы.
- db/ - SQLite-базы данных

**Структура Базы Данных**  
Основная база team_chat.db хранит пользователей и информацию о чатах:
- users - пользователи:  
    id, login, email, hashed_password
- chats - чаты:  
    id, title, is_group, owner_id, messages_db_path, created_date
- chat_members - участники чатов:  
    id, chat_id, user_id

Для каждого чата создается отдельная база сообщений, например chat_1.db. В ней хранятся:
- messages - сообщения:  
    id_message, sender_id, message, time
- attachments - прикрепленные файлы:  
    id, message_id, file_name, file_path, file_size, mime_type

**Особенности Проекта**
Проект разделяет основную базу пользователей и отдельные базы сообщений для каждого чата. Это упрощает хранение сообщений и файлов. Пользователь может удалять и редактировать только свои сообщения, а доступ к чату и скачиванию файлов проверяется через участие пользователя в этом чате.

**Возможности для расширения**  
Проект может стать полноценной социальной сетью с профилями пользователей, каналами, ботами и т.д.
